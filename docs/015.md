# 3. API Principles

## Edge First

Mọi API đều chạy trên Cloudflare Workers.

Không sử dụng server truyền thống.

---

## Read Optimized

Một request

↓

Một response

Không để Frontend gọi 10 API.

---

## Streaming First

API AI

API Export

API Search

đều hỗ trợ Streaming.

---

## Cache Friendly

Mỗi API phải định nghĩa rõ

Cache TTL

ETag

Stale While Revalidate

Invalidate Strategy

---

## Stateless

Worker không lưu Session.

Session lưu bằng JWT hoặc Auth Provider.
